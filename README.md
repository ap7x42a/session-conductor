# Session Conductor

Reusable Agent Skill for substantial sessions that need coordination,
delegation, adversarial review, and handoff discipline.

The conductor keeps judgment. Delegates can gather evidence, perform bounded
mechanical work, or oppose a claim, but the conductor reviews receipts and makes
the final decision.
